<div> Definition and usage

The <div> tag defines a division or a section in an HTML document. The <div> element is often used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript.

<code> Definiton and usage

The <code> tag is a phrase tag. It defines a piece of computer code. Tip: This tag is not deprecated, but it is possible to achieve richer effect with CSS.

<span> Definiton and usage

The <span> tag is used to group inline-elements in a document. The <span> tag provides no visual change by itself. The <span> tag provides a way to add a hook to a part of a text or a part of a document.
